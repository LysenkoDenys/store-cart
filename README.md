A simple **React + Redux Toolkit** app for managing shop items.  
Built with **React 19**, **Redux Toolkit**, **Material UI**, **Vite**, and **TypeScript**.

## 🖥️ Features

- ➕ Add items (name, price, amount)
- 🔍 Search items by name
- 🗑️ Delete items
- 📊 Live summary:
  - total item count
  - total quantity
  - total value
